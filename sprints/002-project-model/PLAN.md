# Sprint 002 - Project Intelligence Engine

## Objective

Build the first intelligent layer of AKYDev.

Instead of simply counting files, AKYDev should understand a software project.

---

## Features

- Detect project type
- Detect Python packages
- Detect entry point
- Count Python files
- Count functions
- Count classes
- Count imports
- Detect dependencies

---

## Success Criteria

Running:

akydev analyze <project>

should provide a detailed overview of the project structure.

---

## Future

This information will become the Project Model used by the AI Planner.
